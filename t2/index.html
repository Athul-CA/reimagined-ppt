$ cd

  • Stands for Change Directory
  • We can use this command to browse between folders in the system

$ ls

  • ls stands for List
  • Lists the info about files

$ pwd

  • pwd stands for Print Working Directory
  • Prints the current working Directory.

$ cat

  • cat displays the content of the files.
  • Syntax $ cat filename.extension

$ mkdir

  • mkdir makes a directory/folder.
  • Syntax $ mkdir newfolder

$ rmdir

  • rmdir deletes a folder/directory.
  • Syntax $ rmdir newfolder
